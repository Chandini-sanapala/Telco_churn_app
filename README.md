# 📊 Telco Customer Churn Prediction using Machine Learning

## 🔍 Project Overview

This project aims to predict whether a customer will churn (leave the company) or not using machine learning techniques. Customer churn is a critical problem in the telecom industry, and predicting it helps businesses take preventive actions to retain customers.

---

## 🎯 Objective

* To build a machine learning model that predicts customer churn
* To analyze customer behavior using data visualization
* To deploy the model using Streamlit for real-time predictions

---

## 📁 Dataset

The dataset contains customer information such as:

* Demographic details (gender, senior citizen, etc.)
* Services used (internet, streaming, phone services)
* Account information (tenure, contract type, charges)
* Target variable: **Churn (Yes/No)**

---

## ⚙️ Technologies Used

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn
* Streamlit
* Pickle

---

## 🧪 Project Workflow

1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Encoding & Scaling
5. Model Building (Logistic Regression, Random Forest, etc.)
6. Model Evaluation (Accuracy, Confusion Matrix, ROC-AUC)
7. Model Deployment using Streamlit

---

## 🤖 Model Performance

* Achieved good accuracy and ROC-AUC score (~0.86)
* Evaluated using classification report and confusion matrix

---

## 🌐 Deployment

The model is deployed using **Streamlit**, allowing users to input customer details and get real-time churn predictions.

## 🚀 How to Run the Project

1. Clone the repository
2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the Streamlit app

```
streamlit run app.py
```

---

## 💡 Key Insights

* Customers with low tenure are more likely to churn
* Higher monthly charges increase churn probability
* Contract type plays a significant role in customer retention

---

## 📌 Conclusion

This project demonstrates how machine learning can be used to predict customer churn and help businesses improve customer retention strategies.
